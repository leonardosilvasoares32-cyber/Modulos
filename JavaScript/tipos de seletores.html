o que sao os elementos de seletores?

getElementByClassName - seleciona todos os elementos com a mesma classe
getElementById - seleciona um elemento pelo id
getElementsByTagName - seleciona elementos pelo nome da tag
getElementByName - seleciona elementos pelo atributo name
querySelector - seleciona o primeiro elemento que corresponde a um seletor CSS especifico
querySelectorAll - seleciona todos os elementos que correspondem a um seletor CSS especifico

esses são os seletores mais utilizados no JavaScript
